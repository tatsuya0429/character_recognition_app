## インストール方法

```
cd charater_recognition_app
npm install
```
frontendでは
```
cd frontend
npm install
```

backendでは、pipenvが必要なのでpipenvをインストールしてください。
```
pip install pipenv
pipenv install
```

## 起動方法
```
cd backend
pipenv run start
```
